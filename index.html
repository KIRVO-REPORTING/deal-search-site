Personal shopping data utility

Deal Search Tool

This site describes a personal product-search utility that compares public shopping data from Rakuten Ichiba and Yahoo Shopping, with a focus on price, discount signals, and point reward rates.

Purpose

The utility retrieves product information to identify items with meaningful discounts or high point rewards for personal comparison and analysis.

Data Used

Product names, prices, URLs, review counts, sale indicators, shipping labels, affiliate rates, and point reward fields returned by the official APIs.

Operation

API requests are made from a private local script. No user accounts, passwords, payment data, or order history are collected on this website.
